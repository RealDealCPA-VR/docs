# Get Current Savings API

**Endpoint:** `/api/get-current-savings`

**Description:**  
Retrieve the current savings information for a specified user.

## Request

- **Method:** GET
- **Query Parameters:**  
  - `userId` (string): The user's unique identifier.

## Response

- **200 OK:** Current savings details.
