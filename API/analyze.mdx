# Analyze Address

**Endpoint:** `POST /analyze`

**Summary:**  
Analyze Address

**Description:**  
Performs a comprehensive real estate investment analysis.

- **Security:** Requires valid API key in `X-API-KEY` header (format: ak_<key_id>.<secret>).
- **Credits:** Charges 10 credits per analysis.
- **Errors:** Returns 401 if invalid key, 402 if insufficient credits.

To test this endpoint in the FastAPI docs (`/docs`), click "Authorize" and enter your API key.

## Request

- **Method:** POST
- **Body:**  
  - `address` (string, required): The address to analyze.

- **Headers:**
  - `X-API-KEY` (string, required): Your API key.

## Responses

- **200 OK:** Successful Response (JSON, see `AnalysisResponse` schema)
- **401 Unauthorized:** Invalid API key
- **402 Payment Required:** Insufficient credits
- **422 Validation Error:** Invalid input

## How to Call This Endpoint

```bash title="cURL Example"
curl -X POST "https://api.example.com/analyze" \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: ak_abc123.secret456" \
  -d '{"address": "123 Main St, Springfield, IL 62704"}'
```
