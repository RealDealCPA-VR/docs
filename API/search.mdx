# Search Address

**Endpoint:** `GET /search`

**Summary:**  
Search Address

**Description:**  
Uses a Google-first strategy to find and scrape property data.

- **Security:** Requires valid API key in `X-API-KEY` header (format: ak_<key_id>.<secret>).
- **Credits:** Charges 5 credits per search.
- **Errors:** Returns 401 if invalid key, 402 if insufficient credits.

To test this endpoint in the FastAPI docs (`/docs`), click "Authorize" and enter your API key.

## Request

- **Method:** GET
- **Query Parameters:**
  - `address` (string, required): Full address to search.

- **Headers:**
  - `X-API-KEY` (string, required): Your API key.

## Responses

- **200 OK:** Successful Response (JSON, see `SearchResponse` schema)
- **401 Unauthorized:** Invalid API key
- **402 Payment Required:** Insufficient credits
- **422 Validation Error:** Invalid input

## How to Call This Endpoint

```bash title="cURL Example"
curl -X GET "https://api.example.com/search?address=123+Main+St,+Springfield,+IL" \
  -H "X-API-KEY: ak_abc123.secret456"
```

## Example Request

```http
GET /search?address=123+Main+St,+Springfield,+IL HTTP/1.1
```
